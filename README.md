## Install

Make sure to install the dependencies:

```bash
# yarn
yarn install

# npm
npm install

# pnpm
pnpm install
```
---
## Development Server
### Vue 3 + TypeScript + Nuxt 3 + TailwindCSS

Start the development server on http://localhost:3000

```bash
# yarn
yarn dev

# npm
npm run dev
```
